# 🏥 Hospital Management System

A simple **Hospital Management System** developed using **Python** and **Pandas**. This project helps manage patient records stored in CSV files through a menu-driven command-line interface.

---

## 📌 Features

- 👨‍⚕️ View all patient records
- 🔍 Search patient by Patient ID
- ➕ Add new patient
- ✏️ Update patient details
- ❌ Delete patient records
- 📅 Manage appointments
- 💾 Save updated records to CSV files
- 📊 Display patient information in a structured format

---

## 🛠️ Technologies Used

- Python
- Pandas
- CSV File Handling

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/Hospital-Management-System.git
```

2. Install the required library.

```bash
pip install pandas
```

3. Open the project folder.

4. Run the program.

```bash
python hospital_management.py
```
